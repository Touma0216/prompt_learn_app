# 🌱 AIプロンプト学習アプリ開発：ピノ

## 🎯 目的

このプロジェクトは、**AI初心者が「プロンプトとは何か」「どう使うのか」を体系的に学べる**スマホ・Webアプリを個人開発しています。  
iOS／Android／Webの3プラットフォームで展開し、教育・娯楽・収益性のバランスをとりながら持続可能な運営を目指します。

## 📐 UI設計の基本方針

- メイン画面はジャンル別カテゴリカード（クリックで詳細に遷移）
- 詳細画面はQiita風の縦スクロールUI（HTML＋CSS表示）＋目次付き
- 目次は「現在表示中のセクションをハイライト」＋クリックで該当見出しへスクロール
- Drawerメニューにて「設定／規約／サブスク管理」などを統合

## 🧸 Pino（ピノ）について

「ピノ」は、本アプリの**マスコットキャラクター**として登場します。  
教育用途を重視するアプリであることから、Live2D等の二次元キャラを導入しづらいシーン（例：企業・学校利用）においても受け入れられるよう、より抽象的で中立的な存在として設計されました。

- 名前の由来は  
  **P**rompt **I**nterface **N**avigation **O**perator（プロンプト案内役）に由来しています。
- アプリ名でもあり、正式名称が未定の段階での**仮名称**でもあります。
- 外見は「幽霊」に近い簡素なシルエットで、  
  「**触れることはできないが、もっとも人に近い存在**」という抽象的な構想がベースです。
- キャラクターの立ち位置としては、Androidのロボットのような**中立的かつ印象的なマスコット**を目指しています。

## 🎀 Live2Dキャラクター構想

本アプリでは、**等身大のLive2Dキャラクター**と**ミニキャラ**を導入予定です。

- キャラ構成は2名：  
  - **清楚系彼女**（メイン）  
  - **イケメン彼氏**（売れ行き次第で追加）

- 実装方針：
  - 初期リリースでは**清楚系彼女のみ実装**
  - いずれもLive2Dで外注
  - ボイスもAI生成音声を使用し、明記の上で学習・編集を行います

- ミニキャラ（常時画面右下に表示）は以下のモーション対応を予定：
  - 待機モーション
  - タップ反応モーション
  - 「つまむ」「移動させる」などのUI演出
  - ユーザーとの**好感度システム**（ミニキャラと遊ぶ機能）  
    → この好感度は、等身キャラとの会話内容・反応にも影響します
